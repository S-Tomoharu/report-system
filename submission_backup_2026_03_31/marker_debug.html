パラメータ説明

UI表示の略記号

略記号 意味 説明
C Contrast 対比度（周囲との明暗差）。高いほど「マーカーらしい」
B Brightness 中央の輝度（0=黒、255=白）。低いほど黒い
D Dark Ratio マーカー内の暗ピクセル比率（%）。高いほど黒く詰まっている
Br Border Brightness マーカー周囲の輝度。高いほど周りが白い

パラメータ設定

パラメータ名 デフォルト 範囲 説明
MIN_CONTRAST 15 0～100 マーカーとして認識する最小対比度
MARGIN_RATIO 0.5 0～2 周囲領域のマージン率（マーカーサイズの何倍か）
MARKER_SIZE_RATIO 0.03 0.01～0.2 マーカーサイズの比率（画像幅に対する%）
EXCLUDE_MARGIN_RATIO 0.03 0～0.3 各隅での除外領域の比率
MAX_CENTER_BRIGHTNESS 120 0～255 中央が「黒い」と判定する最大輝度
DETECTION_RANGE_RATIO 0.2 0.1～1 各隅からの探索範囲（画像サイズの何%まで）
DARK_PIXEL_RATIO_MIN 0.6 0～1 マーカー内の暗ピクセル最小比率（0.6 = 60%以上）
MIN_BORDER_BRIGHTNESS 200 0～255 周囲が「白い」と判定する最小輝度
BORDER_CHECK_DISTANCE 3 1～10 マーカー輪郭から外側何ピクセルで枠をチェックするか
MIN_CANDIDATE_DISTANCE 50 10～500 候補同士の最小距離（ピクセル）。値を上げると分散した候補が得られる

フィルタリング条件

候補が採用されるには以下の条件を全て満たす必要があります：

  • ✓ 対比度 ≥ MIN_CONTRAST
  • ✓ 中央の輝度 ≤ MAX_CENTER_BRIGHTNESS
  • ✓ 暗ピクセル比率 ≥ DARK_PIXEL_RATIO_MIN
  • ✓ 周囲の輝度 ≥ MIN_BORDER_BRIGHTNESS

条件を満たさない候補には理由が表示されます：
(暗不足) = 暗ピクセル比率が不足
(枠暗) = 周囲が十分に白くない
