## 発見された課題・今後の改善点

### 🟡 アップロード時間が長い
- **症状**: ファイルアップロード → 補正 → OCR API 呼び出しで耐えられない
- **原因**:
  - HEIC デコード（libheif-js）のロード時間
  - Canvas 処理（透視変換、OMR 読み取り）
  - 複数ファイル時に処理が順序実行
- **対策案**:
  - UI ローディング表示の改善
  - 非同期処理の最適化
  - 後日、performance profiling で ボトルネック特定

### 🟡 round パラメータの扱い
- **現状**: quiz_prototype_v2.html から submission.html へ遷移時、round を URL に含める必要
- **改善案**:
  - submission.html で sessionStorage から round を自動抽出
  - または、URL パラメータのバリデーション強化

### 🟡 OCR 結果の記録（後日実装）
- submission_gas.gs は markerPageIndex を記録しているだけ
- Claude API でクラス・番号認識後、スプレッドシート「OCR結果」に記録
- **タイミング**: Claude API キーを CLAUDE_API_KEY に設定後に実装
